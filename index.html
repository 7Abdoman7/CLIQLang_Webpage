What is CLIQLang?

CLIQLang is a quantum programming language designed for defining and executing quantum circuits using an intuitive custom syntax. It bridges the gap between theoretical quantum computing concepts and practical implementation.

Perfect for quantum education, research, and algorithm development, CLIQLang supports standard quantum gates, parametric rotations, custom matrix definitions, oracle operations, conditional logic, and loops.

Bell State Creation

Create quantum entanglement between two qubits
